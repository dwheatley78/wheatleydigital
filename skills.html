Digital Skills and Resources

We have collated a list of useful online resources to support a digital service transformation, but if you can't find what you are looking for please feel free to get in touch and we will do our best to help.

  • Free Resources

Free resources

Strategy and Management

  • Business Canvases - A selection of canvases to help visualise complex business issues. We particularly like using the value proposition canvas!
  • Elements of AI - This short course provides a free, non-technical, overview of AI, covering what it can and can't deliver.

Programming and Machine Learning

  • Python First Steps - This course from Microsoft is a good place to start learning Python.
  • Python DataScience Handbook - This free book provides a fantastic introduction to the main Python analytical libraries.
  • Real Python - Once you've got the basics the podcasts and tutorials offered by Real Python will support your development towards becoming a professional developer
  • Machine Learning Mastery - A great source of ideas when developing your own machine learning applications.

Creating a Website

  • Introduction to HTML- Learn fundamental HTML skills and build your first webpage
  • W3 schools - Detailed explanations of the key web development tools (HTML, CSS and JavaScript) .
  • Building a website - great article walking through the process how of create a portfolio website using HTML, CSS, JavaScript and Bootstrap 5
  • React - An introduction to the React framework to build component based single page applications
  • Django - A guide to backend development using Django, a popular python framework
  • Flask Mega Tutorial - Flask offers a lighterweight aThis free tutorial provides an excellent introduction to building web apps using Python.
  • Full Stack Python - Resources to help develop your Python skills to develop web applications

Hosting and Marketing a Website

  • GitHub Pages - This tutorial will teach you the basics about hosting a static website for free
  • Search Engine Optimisation (SEO) - A Google guide to increasing the visibility of your website on search engines
  • Analytics Academy - A free course from Google explaining their tools to enable intelligent data collection and analysis from your website.
  • Google Ads - A guide to setting up an Google Ad campaign to promote your website.

Maths

  • Khan Academy - Linear algebra, statistics and calculus are key to developing machine learning capabilities. If you need to brush up these skills Khan Academy is a great resource.
